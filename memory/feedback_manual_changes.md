O usuário quer fazer todas as mudanças de código manualmente.

**Why:** Preferência pessoal — quer ter controle total sobre o que é alterado no projeto.

**How to apply:** Nunca usar Edit/Write para modificar arquivos do projeto. Apenas mostrar o trecho atual, o que precisa mudar, e onde (arquivo + linha). Deixar a aplicação para o usuário.
